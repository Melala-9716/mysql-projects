## 🏫 Biruk Academy School Management Database

### Description
A multi-branch school management system designed to handle students, teachers, payments, grades, and schedules using a normalized relational database structure.

---

### Database Modules
- Branch Management
- Student Records
- Teacher Records
- Payments System
- Grade Reports
- Class Scheduling

---

### Features
- Multi-branch architecture
- Student–teacher relational mapping
- Payment tracking (Paid / Partial / Overdue)
- Academic performance tracking
- Timetable scheduling system
- Aggregated SQL reports using JOINs and GROUP BY

---

### ER Diagram
![ER Diagram](./diagrams/Biruk%20Academy%20Er%20Diagram.jpg)
---

### Sample Queries Included
- Student + Branch details (JOIN)
- Teacher timetable
- Student performance ranking
- Revenue per branch
- Overdue payments filtering
- Subject-wise performance averages
- Teacher workload analysis
